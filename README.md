## What is FLOQ?

FLOQ is not an iteration on Agile — it is a replacement, purpose-built for a world where:
- AI agents are team members
- Low-code platforms compress development cycles from weeks to hours
- Continuous deployment makes batch delivery obsolete

## Key Stats
- ~80% ceremony reduction vs Agile/Scrum
- 5–9 weeks reclaimed per person per year
- 5 core pillars
- 0 sprints required

## The 5 Pillars

1. **Ship Continuously** — Feature done? It ships immediately.
2. **Unblock Ruthlessly** — Zero tolerance for idle time. Blockers resolved in <2 hours.
3. **Respond to Real Outcomes** — Metric didn't move? Pivot. No hiding behind story points.
4. **Generate with AI** — AI is a team member, not a tool.
5. **Evolve by Signal, Not Schedule** — No signal? No meeting.
